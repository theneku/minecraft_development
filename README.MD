# Minecraft Development

This is my Minecraft development repository. This space serves as a portfolio of my journey, gathering a wide range of projects from my past experiments to my current, ongoing work. Or this is the objective xD. idk what kind of stuff i'm going to uplod here 😅.

This repository is designed to showcase my growth and technical skills. However, the availability of the source code depends entirely on the nature of each project:

*   **Open Source Projects:** For several works, the full source code is available. Feel free to explore, study, and see how the plugins or modifications were structured.
*   **Closed Source Projects:** For the projects I care about the most, or those currently in active development, the source code is **strictly hidden**. 

**Important Notice:** If the code for a specific project is missing, it is a deliberate choice. I value my hard work, and the code is kept private so that vain kids cannot steal it and falsely claim it as their own.

## Navigation

Each folder represents a specific project (past or present). Inside them, you will find dedicated documentation explaining what the plugin or system does, the technologies used, and its current development status.
